[置頂] 店舖後台「新增報價、鎖價異動」即時通知功能上線 NEW
2015/12/31

阿湯自己本身是用倉頡輸入法，所以不管倉頡怎麼改版我一直都用的很習慣，沒有新舊版差異的問題，不過隨著 Windows 的改版，注音的版本似乎也是一改再改，對阿湯來說感受不出來好不好用，但好像不少人最喜歡的還是舊版的ㄅ半注音輸入法，也就是 XP 時代往前的注音，僅管已經推出到 Windows 10，還是有人追著阿湯問舊版注音，其實這早就有解法，而且也適用於 Windows 7 和 8 的系統，還是習慣舊版注音的朋友，就往下來看怎麼加入舊版注音吧。
